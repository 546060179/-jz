Core Capabilities

一套系统 覆盖动效全链路

不是更多的 keyframes，而是一套统一的语言。让设计师、开发者、AI 说同一种话。

7 种基础动效

fade、scale、slide、rotate、blur、flip、collapse 原子化组合出任何交互。

Design Tokens

时长、缓动、距离、意图四维参数，设计规范直接翻译成代码。

四端统一

React + Vue + iOS Swift + Android Kotlin，同一套 API 同一套预设。

弹簧物理

阻尼谐振子引擎，snappy / bouncy / gentle 等 5 种预设，跨端一致。

生命周期

Presence 自动处理进入/退出，支持 sync / wait 两种模式。

编排工具

stagger 交错入场、planSequence 自动计算延迟，复杂编排一行搞定。

无障碍

自动响应 prefers-reduced-motion，全局 motionLevel 可覆盖。

AI-Ready

每个组件一键生成对应端 Prompt，粘贴到 Cursor / Copilot 即用。

18 种预设

scale-fade-in、slide-up-in、blur-fade-in 等开箱即用。
