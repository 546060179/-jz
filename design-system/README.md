# Design System Component Library

基于 Figma 设计稿生成的三端组件库（Web/React、iOS/SwiftUI、Android/Jetpack Compose）。

## 组件列表

| 组件 | 说明 |
|------|------|
| Tag | 标签徽章（New/Hot/Free/Exclusive/Members Only） |
| TabSwitch | 标签页切换（Popular/Categories） |
| BookCard | 大卡片（封面+标签+标题+描述） |
| SmallBookCard | 小卡片（封面+播放量+标题+标签） |
| DateDivider | 日期分割线 |
| RemindButton | 提醒/已预约按钮 |
| UpdateReminderCard | 更新提醒卡片（日期+封面+标题+提醒按钮） |

## 设计 Token

- 主色: `#5A68FF` (Primary)
- 背景深色1: `#050713` (Bg-D1)
- 背景深色2: `#141621` (Bg-D2)
- 文字白: `#FFFFFF`
- 文字灰: `#6C7398`
- 文字浅灰: `#C4C7D6`
- 字体: Lexend Deca
- 圆角: 8px / 12px
